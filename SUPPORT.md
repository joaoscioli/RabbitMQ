# Support

This repository is maintained as a RabbitMQ messaging lab. Questions and
suggestions are welcome when they improve the reliability, observability, or
clarity of the messaging examples.

## Best Ways To Ask For Help

- Open an issue for unclear queue, exchange, retry, or dead-letter behavior.
- Include the affected flow, class, configuration, or documentation section.
- Describe the expected message path and the actual result.

## Good Issue Examples

- A retry example does not explain failure handling clearly.
- A message payload should be simplified or documented.
- A local broker setup step is missing.
- A test should cover duplicate, replay, or dead-letter behavior.

## Out Of Scope

- Production broker operations support.
- Real credentials, private payloads, or company infrastructure.
- Feature requests unrelated to messaging reliability or learning value.
